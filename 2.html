📖 Learn the Theory

What are Operators?

Operators are symbols that tell the compiler to perform specific operations.

Types of Operators:

  • Arithmetic Operators: +, -, *, /, % (modulus gives remainder)
  • Relational Operators: ==, !=, <, >, <=, >= (returns 1 for true, 0 for false)
  • Logical Operators: && (AND), || (OR), ! (NOT)
  • Assignment Operators: =, +=, -=, *=, /=
  • Increment/Decrement: ++ (adds 1), -- (subtracts 1)

Operator Precedence (Highest to Lowest):

  1. ( ) - Parentheses
  2. ++ , -- , ! - Unary operators
  3. * , / , % - Multiplication, Division, Modulus
  4. + , - - Addition, Subtraction
  5. < , > , <= , >= - Relational
  6. == , != - Equality
  7. && - Logical AND
  8. || - Logical OR
  9. = - Assignment

Examples:

#include <stdio.h>
int main() {
    int a = 10, b = 3;
    
    // Arithmetic
    printf("Addition: %d\n", a + b);        // 13
    printf("Modulus: %d\n", a % b);         // 1 (remainder)
    
    // Precedence
    printf("10 + 5 * 2 = %d\n", 10 + 5 * 2);     // 20 (multiplication first)
    printf("(10 + 5) * 2 = %d\n", (10 + 5) * 2); // 30 (parentheses first)
    
    // Relational
    printf("10 > 5 = %d\n", 10 > 5);        // 1 (true)
    
    // Logical
    printf("(10>5) && (3<8) = %d\n", (10>5) && (3<8)); // 1
    
    // Increment
    int x = 5;
    printf("++x = %d\n", ++x);              // 6 (pre-increment)
    printf("x++ = %d\n", x++);              // 6 then becomes 7
    return 0;
}

Key Points to Remember:

  • % only works with integers
  • ++x increments then uses, x++ uses then increments
  • Relational operators return 1 (true) or 0 (false)
  • Parentheses have the highest precedence

✍️ Coding Practice (3 Questions)

Question 1

Write a program to swap two numbers without using a third variable.

#include <stdio.h>
int main() {
    int a = 5, b = 10;
    printf("Before: a=%d, b=%d\n", a, b);
    a = a + b;
    b = a - b;
    a = a - b;
    printf("After: a=%d, b=%d\n", a, b);
    return 0;
}

Question 2

Write a program to check if a number is even or odd using ternary operator.

#include <stdio.h>
int main() {
    int num = 7;
    (num % 2 == 0) ? printf("Even") : printf("Odd");
    return 0;
}

Question 3

Write a program to demonstrate operator precedence: compute 10 + 20 * 30 / 10.

#include <stdio.h>
int main() {
    int result = 10 + 20 * 30 / 10;
    // 20*30=600, 600/10=60, 10+60=70
    printf("Result: %d\n", result);
    return 0;
}
