📖 Learn the Theory

What are Strings in C?

Strings are character arrays terminated by a null character '\0'.

Declaration and Initialization:

  • char str[20]; - Can hold 19 characters + null
  • char name[] = "Hello"; - Size automatically calculated (6 bytes: H,e,l,l,o,\0)
  • char str[5] = {'H','e','l','l','o','\0'}; - Explicit null terminator

String Input/Output:

  • scanf("%s", str); - Reads until whitespace (no & needed)
  • gets(str); - Reads entire line (unsafe, avoid)
  • fgets(str, size, stdin); - Safe way to read string
  • printf("%s", str); - Prints string
  • puts(str); - Prints string and adds newline

Common String Functions (string.h):

  • strlen(str) - Returns length of string (excluding null)
  • strcpy(dest, src) - Copies src to dest
  • strcat(dest, src) - Appends src to dest
  • strcmp(str1, str2) - Compares strings (0 if equal)
  • strchr(str, ch) - Finds first occurrence of character
  • strstr(str, substr) - Finds substring

Examples:

#include <stdio.h>
#include <string.h>
int main() {
    // String length manually
    char str[] = "Hello";
    int len = 0;
    while(str[len] != '\0') len++;
    printf("Length: %d\n", len);  // 5
    
    // Using strlen
    printf("strlen: %lu\n", strlen(str));
    
    // String copy
    char dest[20];
    strcpy(dest, str);
    printf("Copied: %s\n", dest);
    
    // String concatenation
    char str1[50] = "Hello ";
    char str2[] = "World";
    strcat(str1, str2);
    printf("Concatenated: %s\n", str1);
    
    // String comparison
    if(strcmp("abc", "abc") == 0)
        printf("Strings are equal\n");
    
    // Count vowels
    char word[] = "Education";
    int vowels = 0;
    for(int i = 0; word[i] != '\0'; i++) {
        char ch = tolower(word[i]);
        if(ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u')
            vowels++;
    }
    printf("Vowels: %d\n", vowels);  // 5
    
    return 0;
}

Key Points to Remember:

  • Strings always end with '\0' (null character)
  • Don't forget to allocate space for null terminator
  • strcmp returns 0 if strings are equal
  • Use fgets() instead of gets() for safety

Question 1

Write a program to check if a string is palindrome (reads same forwards and backwards).

#include <stdio.h>
#include <string.h>
int main() {
    char str[] = "madam";
    int len = strlen(str);
    int isPalindrome = 1;
    
    for(int i = 0; i < len/2; i++) {
        if(str[i] != str[len-1-i]) {
            isPalindrome = 0;
            break;
        }
    }
    
    if(isPalindrome) printf("%s is Palindrome", str);
    else printf("%s is not Palindrome", str);
    return 0;
}

Question 3

Write a program to reverse a string without using library function.

#include <stdio.h>
int main() {
    char str[] = "Hello";
    int len = 0, temp;
    
    // Find length
    while(str[len] != '\0') len++;
    
    // Reverse
    for(int i = 0; i < len/2; i++) {
        temp = str[i];
        str[i] = str[len-1-i];
        str[len-1-i] = temp;
    }
    
    printf("Reversed: %s", str);
    return 0;
}
