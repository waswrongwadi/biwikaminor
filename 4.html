📖 Learn the Theory

What is an Array?

Array is a collection of elements of the same data type stored in contiguous memory locations.

1D Array (One-Dimensional):

  • Declaration: datatype arrayName[size];
  • Example: int marks[5];
  • Initialization: int marks[5] = {85, 90, 78, 92, 88};
  • Access: marks[0] = first element (index starts at 0)

2D Array (Two-Dimensional):

  • Declaration: datatype arrayName[rows][columns];
  • Example: int matrix[3][4];
  • Initialization: int matrix[2][3] = {{1,2,3},{4,5,6}};
  • Access: matrix[row][col]

Examples:

#include <stdio.h>
int main() {
    // 1D Array - find max and sum
    int arr[5] = {10, 20, 30, 40, 50};
    int sum = 0, max = arr[0];
    
    for(int i = 0; i < 5; i++) {
        sum += arr[i];
        if(arr[i] > max) max = arr[i];
    }
    printf("Sum: %d, Max: %d\n", sum, max);  // Sum: 150, Max: 50
    
    // 2D Array - matrix addition
    int A[2][2] = {{1,2},{3,4}};
    int B[2][2] = {{5,6},{7,8}};
    int C[2][2];
    
    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 2; j++) {
            C[i][j] = A[i][j] + B[i][j];
            printf("%d ", C[i][j]);
        }
        printf("\n");
    }
    // Output:
    // 6 8
    // 10 12
    
    // Finding largest in 2D
    int mat[2][3] = {{3,7,2},{9,4,6}};
    int largest = mat[0][0];
    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 3; j++) {
            if(mat[i][j] > largest) largest = mat[i][j];
        }
    }
    printf("Largest: %d\n", largest);  // Largest: 9
    
    return 0;
}

Key Points to Remember:

  • Array index starts from 0 to size-1
  • No bounds checking in C - be careful!
  • Array name represents the base address (address of first element)
  • 2D arrays are stored in row-major order
