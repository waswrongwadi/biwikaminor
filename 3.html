🔄 Conditionals & Loops

if-else, switch, for, while, do-while, break, continue

📖 Learn the Theory

Conditional Statements:

  • if statement: Executes code block if condition is true
  • if-else: Executes one block if true, another if false
  • else-if ladder: Checks multiple conditions in sequence
  • switch statement: Multi-way decision based on value (works with integers and characters)

Loop Structures:

  • for loop: for(initialization; condition; increment) { } - Used when number of iterations is known
  • while loop: while(condition) { } - Condition checked before execution
  • do-while loop: do { } while(condition); - Executes at least once, then checks condition

Jump Statements:

  • break: Exits the loop or switch immediately
  • continue: Skips the rest of current iteration and moves to next

Examples:

#include <stdio.h>
int main() {
    // if-else example
    int marks = 75;
    if (marks >= 90) {
        printf("Grade A\n");
    } else if (marks >= 75) {
        printf("Grade B\n");  // Output: Grade B
    } else {
        printf("Grade C\n");
    }
    
    // switch example
    int day = 3;
    switch(day) {
        case 1: printf("Monday"); break;
        case 2: printf("Tuesday"); break;
        case 3: printf("Wednesday"); break;  // Output: Wednesday
        default: printf("Other day");
    }
    
    // for loop - print 1 to 5
    printf("\nFor loop: ");
    for(int i = 1; i <= 5; i++) {
        printf("%d ", i);  // 1 2 3 4 5
    }
    
    // while loop - even numbers
    printf("\nWhile loop: ");
    int i = 2;
    while(i <= 10) {
        printf("%d ", i);
        i += 2;  // 2 4 6 8 10
    }
    
    // do-while - runs at least once
    printf("\nDo-while: ");
    int j = 1;
    do {
        printf("%d ", j);
        j++;
    } while(j <= 5);  // 1 2 3 4 5
    
    // break and continue
    printf("\nBreak example: ");
    for(int k = 1; k <= 5; k++) {
        if(k == 3) break;
        printf("%d ", k);  // 1 2
    }
    
    printf("\nContinue example: ");
    for(int k = 1; k <= 5; k++) {
        if(k == 3) continue;
        printf("%d ", k);  // 1 2 4 5
    }
    
    return 0;
}

Key Points to Remember:

  • switch only works with int, char, enum - NOT float or string
  • Don't forget break in switch (otherwise fall-through happens)
  • do-while always executes at least once
  • Infinite loop: for(;;) or while(1)

✍️ Coding Practice (3 Questions)

Question 1

Write a program to check if a number is prime or not.

#include <stdio.h>
int main() {
    int n = 17, i, isPrime = 1;
    for(i = 2; i <= n/2; i++) {
        if(n % i == 0) {
            isPrime = 0;
            break;
        }
    }
    if(isPrime) printf("%d is Prime", n);
    else printf("%d is Not Prime", n);
    return 0;
}

Question 2

Write a program to print multiplication table of any number using for loop.

#include <stdio.h>
int main() {
    int num = 7;
    for(int i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", num, i, num * i);
    }
    return 0;
}

Question 3

Write a program to find factorial using while loop.

#include <stdio.h>
int main() {
    int n = 5, fact = 1, i = 1;
    while(i <= n) {
        fact *= i;
        i++;
    }
    printf("Factorial of %d = %d", n, fact);
    return 0;
}
