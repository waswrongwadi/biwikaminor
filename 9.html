🎯 Pointers

Pointer Definition, Operations, Array of Pointers, Self-referential Structures, Call by Reference

📖 Learn the Theory

What is a Pointer?

A pointer is a variable that stores the memory address of another variable.

Pointer Syntax:

  • Declaration: datatype *ptr_name;
  • Address of operator (&): Returns memory address
  • Dereference operator (*): Accesses value at address

Pointer Operations:

  • Assignment: ptr = &var;
  • Dereferencing: *ptr = value;
  • Pointer Arithmetic: ptr++ (moves to next element), ptr--
  • Comparison: ptr1 == ptr2

Array of Pointers:

int *ptr_arr[10];  // Array of 10 integer pointers
char *names[5] = {"John", "Jane", "Bob", "Alice", "Tom"};

Self-referential Structures:

Structures that contain a pointer to the same structure type - used for linked lists, trees.

struct Node {
    int data;
    struct Node *next;  // Pointer to same structure
};

Call by Reference using Pointers:

Passing addresses to functions allows modification of original variables.

Examples:

#include <stdio.h>

// Self-referential structure for linked list
struct Node {
    int data;
    struct Node *next;
};

// Call by reference - swap function
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    // Basic pointer
    int num = 42;
    int *ptr = #
    
    printf("Value: %d\n", num);
    printf("Address: %p\n", &num);
    printf("Pointer stores: %p\n", ptr);
    printf("Dereferenced: %d\n", *ptr);
    
    // Modify through pointer
    *ptr = 99;
    printf("Modified value: %d\n", num);
    
    // Pointer arithmetic with arrays
    int arr[] = {10, 20, 30, 40, 50};
    int *p = arr;  // Points to first element
    
    for(int i = 0; i < 5; i++) {
        printf("*(p+%d) = %d\n", i, *(p + i));
    }
    
    // Array of pointers to strings
    char *fruits[] = {"Apple", "Banana", "Cherry", "Date"};
    for(int i = 0; i < 4; i++) {
        printf("%s\n", fruits[i]);
    }
    
    // Call by reference
    int x = 10, y = 20;
    printf("\nBefore swap: x=%d, y=%d\n", x, y);
    swap(&x, &y);
    printf("After swap: x=%d, y=%d\n", x, y);
    
    // Self-referential structure example
    struct Node n1, n2;
    n1.data = 100;
    n1.next = &n2;
    n2.data = 200;
    n2.next = NULL;
    
    printf("\nLinked list node1: %d, node2: %d\n", n1.data, n1.next->data);
    
    return 0;
}

Key Points to Remember:

  • Pointer size is same for all types (8 bytes on 64-bit system)
  • NULL pointer points to nothing (dereferencing causes crash)
  • Array name is a constant pointer to first element
  • Self-referential structures are building blocks for data structures

Question 1

Write a program to find the length of a string using pointers (without strlen).

#include <stdio.h>
int stringLength(char *str) {
    int len = 0;
    while(*str != '\0') {
        len++;
        str++;
    }
    return len;
}
int main() {
    char str[] = "Hello World";
    printf("Length: %d", stringLength(str));
    return 0;
}

Question 2

Reverse an array using pointers.

#include <stdio.h>
void reverseArray(int *arr, int size) {
    int *start = arr;
    int *end = arr + size - 1;
    int temp;
    
    while(start < end) {
        temp = *start;
        *start = *end;
        *end = temp;
        start++;
        end--;
    }
}
int main() {
    int arr[] = {1,2,3,4,5};
    int size = 5;
    
    reverseArray(arr, size);
    
    for(int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    return 0;
}

Question 3

Create a self-referential structure for a simple linked list node and demonstrate.

#include <stdio.h>

struct Node {
    int data;
    struct Node *next;
};

int main() {
    struct Node n1, n2, n3;
    struct Node *head = &n1;
    
    n1.data = 10;
    n1.next = &n2;
    
    n2.data = 20;
    n2.next = &n3;
    
    n3.data = 30;
    n3.next = NULL;
    
    // Traverse linked list
    struct Node *current = head;
    while(current != NULL) {
        printf("%d -> ", current->data);
        current = current->next;
    }
    printf("NULL\n");
    
    return 0;
}
