📖 Learn the Theory

What is a Function?

A function is a block of code that performs a specific task. It promotes code reusability and modularity.

Function Syntax:

return_type function_name(parameter_list) {
    // function body
    return value;
}

Function Components:

  • Return type: Data type of value returned (void if nothing returned)
  • Function name: Unique identifier
  • Parameters: Inputs to the function (optional)
  • Function body: Code that executes

Call by Value vs Call by Reference:

  • Call by Value: Copy of variable passed. Changes inside function do NOT affect original.
  • Call by Reference: Address of variable passed using pointers. Changes affect original.

Scope and Lifetime:

  • Local variables: Declared inside function, accessible only within that function
  • Global variables: Declared outside all functions, accessible everywhere
  • Static variables: Retain value between function calls

Passing Arrays to Functions:

Arrays are always passed by reference (address is passed, not copy).

Examples:

#include <stdio.h>

// Function prototype
int add(int a, int b);
void swapByValue(int a, int b);
void swapByReference(int *a, int *b);
int sumArray(int arr[], int size);
void printMessage();

// Global variable
int global = 100;

int main() {
    // Call by value
    int x = 10, y = 20;
    printf("Add: %d\n", add(x, y));  // 30
    
    // Swap demonstration
    printf("\nBefore swap: x=%d, y=%d\n", x, y);
    swapByValue(x, y);
    printf("After swapByValue: x=%d, y=%d (no change)\n", x, y);
    swapByReference(&x, &y);
    printf("After swapByReference: x=%d, y=%d (changed!)\n", x, y);
    
    // Array to function
    int arr[] = {1,2,3,4,5};
    printf("\nSum of array: %d\n", sumArray(arr, 5));
    
    // Static variable example
    printMessage();
    printMessage();
    printMessage();
    
    return 0;
}

// Function definitions
int add(int a, int b) {
    return a + b;
}

void swapByValue(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

void swapByReference(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int sumArray(int arr[], int size) {
    int sum = 0;
    for(int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return sum;
}

void printMessage() {
    static int count = 0;  // Static - retains value
    count++;
    printf("Called %d times\n", count);
}

Key Points to Remember:

  • Function prototype is required before calling (or define before main)
  • Arrays are automatically passed by reference (no & needed)
  • Static variables are initialized only once
  • void functions don't return anything

✍️ Coding Practice (3 Questions)

Question 1

Write a function to check if a number is Armstrong (sum of cubes of digits equals number).

#include <stdio.h>
int isArmstrong(int n) {
    int original = n, sum = 0, digit;
    while(n > 0) {
        digit = n % 10;
        sum += digit * digit * digit;
        n /= 10;
    }
    return (sum == original);
}
int main() {
    int num = 153;
    if(isArmstrong(num)) printf("%d is Armstrong", num);
    else printf("%d is not Armstrong", num);
    return 0;
}

Question 2

Write a function that accepts an array and returns the sum of even elements.

#include <stdio.h>
int sumEven(int arr[], int size) {
    int sum = 0;
    for(int i = 0; i < size; i++) {
        if(arr[i] % 2 == 0) sum += arr[i];
    }
    return sum;
}
int main() {
    int arr[] = {1,2,3,4,5,6};
    printf("Sum of evens: %d", sumEven(arr, 6));
    return 0;
}

Question 3

Write a program demonstrating static variable - count how many times function is called.

#include <stdio.h>
void counter() {
    static int count = 0;
    count++;
    printf("Function called %d times\n", count);
}
int main() {
    counter();
    counter();
    counter();
    return 0;
